# Evaluador Académico

Aplicación web desarrollada con HTML, CSS (Bootstrap) y JavaScript que permite evaluar el rendimiento académico de estudiantes.

## Funcionalidades implementadas

1. Formulario con 4 notas, nombre y carrera
2. Notas con decimales (step="0.01")
3. Carreras: TI, Software, Sistemas
4. Cálculo de beca según carrera y promedio
5. Colores según tipo de beca (verde, azul, amarillo, gris)
6. Promedio especial sin la nota más baja
7. Nota más alta y nota más baja
8. Conteo de notas aprobadas y no aprobadas
9. Recomendación académica personalizada
10. Clasificación de rendimiento (Alto/Medio/Básico/Bajo)
11. Historial de estudiantes evaluados
12. Validación de nombres repetidos
13. Botón para listar estudiantes
14. Total de estudiantes evaluados
15. Conteo de aprobados, recuperación y reprobados
16. Promedio general del curso
17. Búsqueda de estudiante por nombre
18. Ranking de estudiantes
19. Mejor y peor promedio del historial
20. Objeto estudiante completo con todos los campos

## Cómo ejecutar

1. Abrir el archivo `index.html` en un navegador web
2. Llenar el formulario con nombre, carrera y 4 notas
3. Hacer clic en "Evaluar" para procesar los datos
4. Usar "Ver Historial" para ver todos los estudiantes
5. Usar "Ver Ranking" para ver el ordenamiento
6. Usar la búsqueda para encontrar estudiantes por nombre

## Tecnologías

- HTML5
- CSS3 (Bootstrap 5.3.3)
- JavaScript (ES6+)
